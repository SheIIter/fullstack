## Shllter gradio 실행 방법
### 설치 pip
1. Github Pull 진행
2. venv(가상환경) 설정
2-1. 설치 code
python -m venv venv 
2-2. 실행 code
source venv/bin/activate  
3. PIP(패키지) 설정
3-1. pip 업그레이드
pip install --upgrade pip 
3-2. pip 설치
pip install gradio requests python-dotenv langchain-upstage langchain-core
4. Gradio 실행
5. python shelter_gradio.py
